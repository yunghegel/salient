Salient is a 3D editor designed to facilitate the creation and manipulation of 3D scenes and assets. It provides a user-friendly interface and a robust set of tools for developers to build interactive 3D applications.

Most simply, Salient is an editor for creating, editing, and managing 3D scenes which can be exported and used in 3D applications or games. It supports various 3D formats and comes with its own custom renderer and asset management system.

## Key Features

- **Scene Management**: Create and manage multiple scenes within a project.

- **3D Object Manipulation**: Add, transform, and modify 3D objects in the scene.

- **Material and Texture Editing**: Apply and edit materials and textures on 3D models.

- **Lighting and Camera Control**: Set up and adjust lighting and camera perspectives.

- **Asset Management**: Import, organize, and manage 3D assets and resources.

- **Scripting Support**: Add custom behavior to objects using Kotlin scripts.

- **User Interface**: Intuitive UI with panels for scene hierarchy, properties, and asset browsing.


## Technology Stack

- **Kotlin**: The primary programming language used for development.
- **LibGDX**: A powerful framework for 2D and 3D game development, providing rendering, input handling, and asset management capabilities.
- **JSON**: Used for configuration files and scene serialization.
- **Gradle**: Build automation tool for managing dependencies and building the project.
- **Git**: Version control system for source code management.
- **OpenGL**: Graphics API for rendering 3D graphics.
- **GLSL**: Shading language for writing custom shaders.
